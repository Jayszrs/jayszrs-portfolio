# Page Files

Next.js App Router mewajibkan nama file route tetap `app/**/page.js`.
Supaya gampang dibaca, isi halaman dipindah ke file bernama jelas di folder ini.

- `HomePage.js` -> `/`
- `TentangPage.js` -> `/tentang`
- `PengalamanPage.js` -> `/pengalaman`
- `ProyekPage.js` -> `/proyek`
- `PencapaianPage.js` -> `/pencapaian`
- `KontakPage.js` -> `/kontak`
- `AdminPageClient.js` -> `/admin`

Kalau mau edit tampilan halaman, buka file di folder ini. File `app/**/page.js`
cuma wrapper tipis agar routing Next.js tetap jalan.
